## 1. OddFi 是什么

OddFi 是部署在 BNB Smart Chain（BSC）上的去中心化体育预测市场。连接钱包即可使用，无需注册、无需邮箱、无需 KYC，资金由用户自托管。

| 项目 | 内容 |
|---|---|
| 所在链 | BNB Smart Chain（BSC，chainId = 56） |
| 平台代币 | ODD（BEP-20，总量 1 亿枚，不增发） |
| 登录方式 | 钱包连接，无需 KYC |
| 结算方式 | 双源核实 + 多签确认，链上执行 |
| 支持充值 | 多种链（暂不支持 TRX/Tron） |

### 为什么选择 OddFi

传统预测市场存在三个结构性问题，OddFi 从底层重构了资金模型与收益分配：

| 痛点 | 传统方案 | OddFi 方案 |
|---|---|---|
| 滑点严重 | AMM 定价，大额预测滑点高 | Betting Pool 统一对手方，固定回报倍率接单 |
| 资金闲置 | 待结算资金完全锁死 | 仓位借贷：用待结算仓位抵押，借出额度继续参与 |
| 收益垄断 | 平台独占全部利润 | 任何人可向 Betting Pool 注资，按份额分享平台 Margin |
| 代币无锚定 | 代币与业务脱钩 | ODD 买卖税机制让代币价值与平台业务深度挂钩 |

### 与传统中心化体育预测平台的对比

| 维度 | OddFi | 传统平台 |
|---|---|---|
| 账户 | 仅需钱包连接 | 邮箱 + KYC + 身份验证 |
| 资金托管 | 智能合约托管，平台无法单方面提取 | 平台持有你的资金 |
| 结算 | 链上可验证 | 信任运营方 |
| 透明度 | 回报倍率推导过程公开，Margin 结构透明 | 定价是黑箱 |
| 领取 | ≤ $5,000 即时到账 | 人工审核，1–5 天 |
| 收益分配 | 做市用户共享 Margin | 收益归平台 |
