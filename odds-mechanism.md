# 赔率机制

### 赔率怎么来的

平台从第三方专业赔率数据源获取参考倍率，去除原始 Margin 后还原真实概率，再加入平台 10% Margin 生成开盘倍率。

```
回报倍率 = 1 / （真实概率 × Overround 因子）
```

典型 Overround：110%。

### 赔率怎么读

| 赔率 | 隐含概率 | 含义 |
|---|---|---|
| 1.5 | 66.70% | 强势热门 — 低回报 |
| 2.0 | 50.00% | 五五开 — 2 倍投入金额 |
| 3.0 | 33.30% | 冷门方 — 3 倍投入金额 |
| 5.0 | 20.00% | 大冷门 — 5 倍投入金额 |

### 动态调整

每笔下注后，系统自动检测各方向的累计金额差异并动态调整：

- **重仓方向**：倍率压低（降低平台风险）
- **轻仓方向**：倍率上浮（吸引反向下注，平衡敞口）

倍率在开赛前 1 小时锁定，此后不再变化。变动幅度超过 0.05 时实时推送至前端。

### 倍率变动确认

从填写金额到确认期间倍率变动超过 0.05，系统弹窗提示最新倍率，需重新确认。锁定倍率以用户确认时为准。
