### 代币用途

| 用途 | 说明 |
|---|---|

| 奖励 | 首充奖励、推荐奖励、空投 |

### 买卖税

#### 买入税：5%

用户在 PancakeSwap 买入 ODD 时触发，全部作为推荐人质押奖励（自动质押 7 天）。推荐人未质押或无推荐人时转入 StakingPool。

#### 卖出利润税：25%（仅对盈利部分）

```
卖出利润 = 卖出金额 − （卖出数量 × 持仓均价）
亏损卖出：不征税
盈利卖出：对利润部分收取 25%
```

税收流向：

| 去向 | 比例（占利润税） | 说明 |
|---|---|---|
| Token Pool（PancakeSwap LP） | 5pp | 自动注入 ODD/USDT 交易对，LP Token 永久锁仓 |
| StakingPool 奖励池 | 9pp | 分配给所有活跃质押用户 |
| 平台收益 | 6pp | 转入平台地址 |
| Betting Pool | 4pp | 自动补充对赌池资金 |
| 销毁 | 1pp | 转入 `0xdead` 地址，永久销毁 |

### 增长飞轮

```
更多用户下注
  ↓
ODD 需求增加 → DEX 买入 → 推荐人获奖励 → 推荐更多用户
  ↓
ODD 价格上涨 → 持仓产生利润 → 卖出缴纳利润税
  ↓
质押池 / Betting Pool / Token Pool 持续获得税收注入
  ↓
Betting Pool 扩大 → 可支撑更大投注 → 吸引大额用户
  ↓
平台 Margin 增加 → LP 收益提升 → 吸引更多做市流动性
```

### 通缩机制

- **卖出利润税 1pp**：每笔盈利卖出自动销毁至 `0xdead`
- **季度回购销毁**：基于平台收入定期回购 ODD 销毁
- **特殊事件销毁**：如预测池结余销毁
